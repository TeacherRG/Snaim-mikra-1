# Shnaim-mikra-1

Статическое SPA-приложение для чтения главы **«Шлах»** в формате:
- Микра (иврит) — 2 раза
- Таргум — 1 раз

С поддержкой:
- RTL-отображения
- кастомного аудиоплеера
- караоке-подсветки слов
- клика по слову для перехода в аудио
- изменения скорости: `1.0x`, `1.25x`, `1.5x`, `1.75x`, `2.0x`

## Файлы ассетов

По умолчанию приложение использует:
- текст: `./פרשת שלח _ שניים מקרא ואחד תרגום`
- аудио: `./Record-6.mp3`

Если хотите подключить другие локальные файлы, измените пути в:
`/home/runner/work/Snaim-mikra-1/Snaim-mikra-1/TeacherRG/Snaim-mikra-1/src/config.js`

```js
textSource: './ваш_файл_текста',
audioSource: './ваш_файл_аудио.mp3',
```

Также можно задать ручные тайминги слов в `manualWordTimings`:

```js
manualWordTimings: [
  { word: 'וַיְדַבֵּר', start: 0.5, end: 1.2 },
];
```

Архитектура уже поддерживает этот режим, поэтому можно заменить автоматическую интерполяцию без переписывания плеера.

## Запуск

Запустите локальный HTTP-сервер из корня репозитория (важно: не открывать `index.html` через `file://`):

```bash
cd /home/runner/work/Snaim-mikra-1/Snaim-mikra-1/TeacherRG/Snaim-mikra-1
python3 -m http.server 8000
```

Откройте в браузере:

`http://localhost:8000`

## Структура модулей

- `src/textParser.js` — загрузка и разбор текстового файла
- `src/timingEngine.js` — интерполяция таймингов + поддержка JSON-таймингов
- `src/audioController.js` — управление аудио, таймлайн, скорость
- `src/uiRenderer.js` — рендер текста и интерактивные слова
- `src/main.js` — связывание модулей приложения
